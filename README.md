# Bible Verse Scroller 📖✨

A beautiful, modern React web application for discovering, reading, and reflecting on Bible verses. It acts like an infinite scroll feed but for scripture, designed to give users a peaceful and focused environment.

## 🌟 Key Features

- **Infinite Verse Scroller**: Continuously discover random verses from the whole Bible or filter them by specific thematic categories (e.g., Love, Joy, Peace, Faith).
- **Thematic Navigation**: Explore verses curated by topics.
- **Favorites System**: Save and easily access verses that speak to you in a dedicated Favourites tab.
- **Personal Reflections**: Write and save your personal journal entries or reflections tied directly to specific verses.
- **Beautiful UI**: Modern, dark-themed, and responsive interface with an emphasis on typography and an engaging reading experience.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: Tailwind CSS + Custom CSS (with Lucide React icons)
- **Data**: Powered by the [Bible API](https://bible-api.com/) for pulling random sets of verses (King James Version).

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sfeng232/bible-verse-scroller.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd bible-verse-scroller
   ```
3. **Install the dependencies:**
   ```bash
   npm install
   ```
4. **Run the development server:**
   ```bash
   npm run dev
   ```
   *The app should now be running locally at `http://localhost:5173/`.*
